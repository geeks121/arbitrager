# Arbitrager
Automatically trade with arbitrage on the Bitcoin exchanges. 

## Install
```
bundle install --path vendor/bundle
```

## Usage
Please modify default_config.yml and rename config.yml. Then exec arbitrager.rb
```
cd src
ruby arbitrager.rb
```

## Valid exchanges
- Coincheck
- Liquid
